## 核心科学问题

符号回归（PySR）在引力波瞬态噪声（glitches）和探测器高斯噪声背景下，能否可靠地从数据中还原已知的波形物理定律？其在什么信噪比（SNR）水平下会彻底失效？

双黑洞并合产生的引力波波形由广义相对论精确预言的，其主导的inspiral阶段遵循“啁啾”关系，相位演化有严格的解析近似（后牛顿展开），这为我们提供了“真实答案”。

引力波数据既有宽带的高斯噪声（探测器热噪声、量子噪声），也有非高斯、非稳态的瞬态噪声（glitches）。

PySR是学术玩具，发现不了新物理: 过拟合、无法处理噪声、工业落地差; 90年代遗传编程的Julia换皮; 只能在小数据场景下作为探索工具

## 实验路径设计

- [] 噪声鲁棒性基准测试: 证明在高斯噪声下，PySR对相位演化公式的恢复能力随信噪比下降而崩溃。
- [] 引入真实探测器噪声: 展示在真实非白噪声（LIGO敏感曲线形状）下，符号回归可能过拟合到某个特定频率的噪声线。
- [] 证明直面Glitch的过拟合: 证明在存在瞬态大噪声（glitch）时，符号回归会疯狂地过拟合到一个毫无物理意义的复杂函数。
